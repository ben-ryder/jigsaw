    # Breakpoints
    Breakpoints can be used to alter the layout and styling of components
    at different screen sizes. In Jigsaw, components are designed
    mobile-first which means the mobile styling should be created first
    and then breakpoints can be used to change the styling as the screen
    width increases.

    CSS variables can't be used in media queries yet, however if you use
    tools such as [Lightning CSS](https://lightningcss.dev/transpilation.html#custom-media-queries) like Jigsaw does, you're able to
    transpiling the draft spec.
