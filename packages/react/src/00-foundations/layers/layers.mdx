    # Layers
    Jigsaw generally has quite a flat design, however elevation & depth  can still be conveyed through the use of
    different colour shades. Different shades can be used to mimic how light would work in the real world, so elements
    that appear over the top or "nearer" to the user would use a lighter shade.

    Shadows can also be used to improve this effect and z0index can be
    used to actually order things correctly.
